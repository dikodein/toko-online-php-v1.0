============================================
        Web Toko Online PHP & MySQL
============================================

Deskripsi:
-----------
Proyek ini adalah sebuah sistem toko online berbasis web yang dibuat menggunakan PHP dan MySQL.

Fitur Utama:
------------
1. Manajemen Produk - Tambah, edit, dan hapus produk.
2. Keranjang Belanja - Menambahkan produk ke keranjang dan melihat total harga. (Proses)
3. Checkout - Proses pembelian dengan konfirmasi pesanan. (Proses)
4. Manajemen Pengguna - Registrasi, login, dan autentikasi pengguna. (Proses)
5. Admin Panel - Mengelola pesanan, produk, dan pengguna.
6. Pencarian Produk - Memungkinkan pengguna mencari produk berdasarkan kata kunci. (Proses)
7. Metode Pembayaran - Mendukung berbagai metode pembayaran. (Proses)

Teknologi yang Digunakan:
-------------------------
- Backend: PHP
- Database: MySQL
- Frontend: HTML, CSS, JavaScript (Bootstrap untuk tampilan responsif)
- Libraries: jQuery, AJAX

Instalasi:
----------
1. Unduh kode sumber atau clone repository ini.
2. Buat database di MySQL dengan nama `toko-online-v0.1`.
3. Import file SQL yang terdapat dalam folder `database/toko-online-v0.1.sql`.
4. Konfigurasi database
   - Edit file `koneksi.php`.
   - Sesuaikan host, username, password, dan nama database.
5. Jalankan di localhost
   - Gunakan XAMPP/MAMP/WAMP dan pastikan Apache serta MySQL sudah berjalan.
   - Akses di browser melalui `http://localhost/toko-online-v0.1/`.

Cara Menggunakan:
-----------------
- Pengguna dapat mendaftar dan login untuk berbelanja. (Proses)
- Admin dapat mengakses panel admin melalui `http://localhost/toko-online-v0.1/adminpanel/`.
- Produk dapat ditambahkan oleh admin, dan pengguna dapat melihat serta membeli produk yang tersedia.

Lisensi:
--------
